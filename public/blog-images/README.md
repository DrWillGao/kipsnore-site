# Blog images

Everything visual for the Kip blog lives here. Two folders:

- **`diagrams/`** — the original, hand-built SVG diagrams used in posts (auto-exported, cited in each post's caption). Safe to open in a browser to review.
- **`photos/`** — real, licensed photos for article hero images. **This is where you drop photos for me to use.**

## Adding photos (for Will)

1. Drop a JPG, PNG or WebP into `photos/`.
2. Name it after the post slug, e.g. `how-to-stop-snoring.jpg`, `snoring-in-men.jpg`. The slug is the bit in the post URL (`/blog/<slug>/`).
3. Tell me, or I'll pick it up next pass and wire it in as that post's hero image (frontmatter `heroImage: /blog-images/photos/<file>`).

Licensing: please only add photos you have the right to use commercially (your own, or from Unsplash / Pexels, which allow commercial use). No AI-generated images.

## Reviewing everything

Open **/library/** on the site (e.g. the preview URL) to see every diagram and photo in one place.
